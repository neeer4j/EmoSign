# ASL Hand Reference Images

This folder should contain hand images demonstrating each ASL sign.

## Naming Convention:
- Letters: `A.png`, `B.png`, ..., `Z.png`
- Numbers: `0.png`, `1.png`, ..., `9.png`  
- Words: `hello.png`, `thanks.png`, etc.

## Image Requirements:
- Clear hand position on neutral background
- 300x300 pixels recommended
- PNG format with transparency preferred

## Temporary Solution:
If images are not available, the app will display ASCII art representations
and emoji-based hand guides.
